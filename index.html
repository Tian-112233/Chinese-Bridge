还没部署 Worker？
1. 注册 Cloudflare → R2 → 创建存储桶 → 开启公开访问
2. Workers & Pages → 创建 Worker → 粘贴 Worker 代码 → Deploy
3. Worker Settings → Bindings → 绑定 R2，变量名 MY_BUCKET
4. Variables → 添加 AUTH_KEY（你的密码）和 PUBLIC_URL（R2公开域名）
5. 填写上方表单即可开始使用
